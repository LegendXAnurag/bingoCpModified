# Security Policy

## Supported Versions

We only support the latest version of CP Games. Please update to the latest version to receive security updates.

## Reporting a Vulnerability

We take the security of our project seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, please report it via one of the following methods:

- Email the maintainers directly (check the GitHub profile for contact info)
- Open a private security advisory on GitHub

We will acknowledge your report within 48 hours and provide a timeline for a fix.
